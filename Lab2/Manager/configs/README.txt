В корне config файлы располагаются по следующему принципу:
pipelineX - X-ая сессия пайплайна (соответствует заданиям)
exe_configs - конфиги собственных экзекьюторов
encoded_sources - входной и выходной файлы, открывающиеся с кодировкой cp1251 (корректное отображение кириллицы)
default_sources - входной и выходной файлы для стороннего экзекьюторов
